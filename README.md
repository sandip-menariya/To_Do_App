# To_Do_App
This repo contains basic To Do Application built in Django which handle's user's to lists and user can manage them like CRUD operations.
To run this application you need to create a djnago project and in django project -
1.) you will need to replace your project files with this application's project_files folder's specific files and in settings.py you will need to register your application as my application is task_manager in settings.py 
2.) in django project you will need to start new application and replace your app's specific files mentioned with these files mentioned  in App_files then you can start with application.
